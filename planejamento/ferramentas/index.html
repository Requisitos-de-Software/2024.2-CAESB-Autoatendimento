    Introdução

      No desenvolvimento do projeto, diversas ferramentas foram utilizadas para facilitar o fluxo de trabalho e garantir a qualidade do produto final. As ferramentas utilizadas para a realização do projeto estão apresentadas na Tabela 1.

    Ferramentas Utilizadas

    Tabela 1 - Ferramentas

    Ferramenta Finalidade
    Github Organizar, versionar o projeto.
    Whatsapp Comunicação entre os participantes do grupo
    Youtube Hospedagem dos vídeos produzidos
    VScode Edição de arquivos
    Teams Realizações de reuniões e gravações de apresentações.
    Filmora Edição de vídeo
    Mkdocs Criação do gitPage
    Draw.io Utilizado para fazer o Rich Picture
    Figma Utilizado para fazer os protótipos
